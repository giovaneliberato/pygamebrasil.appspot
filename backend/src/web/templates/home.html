PyGame

PyGame (http://www.pygame.org) é um módulo Python que fornece a API da biblioteca SDL (feita em C) e mais algumas funcionalidades para a programação gráfica, em especial jogos. Provê facilidades tais como Sprites, Render Groups, deteção de colisão básica (retângulos) e muito mais. É multi plataforma! Funciona em Linux, Windows, Sun, MacOS, BeOS, ...

http://www.python.org.br/wiki/PyGame

Pygame no android

Para quem gosta de programação de jogos para mobile, mais especificadamente para Android existe um módulo que permite desenvolver jogos para Android com Pygame.

http://pygame.renpy.org/
