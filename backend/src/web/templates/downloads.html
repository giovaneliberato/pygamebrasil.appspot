Distribuições Unix

A versão 1.9.1 está na maioria das distribuições Unix.Você também pode instalar a versão 1.9.1 a partir do código fonte usando executando o arquivo setup.py.

Distribuição
Ubuntu O pygame está disponível via apt-get in the Universe (1.9.1)
Gentoo O pygame está disponível no the portage system (1.9.1 + 1.9.2 pre-release)
Fedora Os repositórios de pacotes possuem suporte ao pygame (1.9.1)
Suse O sistema de pacotes Yast possui o pygame (updated to 1.9.1)
OLPC Vem com o XO, e o sugar (1.9.1)
archlinux O pygame está disponível através do pacman (1.9.1)
Debian O pygame está disponível via apt-get (1.9.1)
FreeBSD O pygame está incluso nas portas padrão como py26-game (1.8.1)
OpenBSD O pygame está incluso nas portas padrão (1.8.1)
